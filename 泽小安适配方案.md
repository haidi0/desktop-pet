# 泽小安 × VPet 适配方案

## 一、项目目标

基于 VPet（虚拟桌宠模拟器）框架，适配"泽小安"角色，打造专属桌面宠物。

## 二、VPet 项目结构分析

```
VPet/
├── VPet-Simulator.Core/          # 核心库（可嵌入其他WPF应用）
│   ├── Handle/                   # 接口与控件
│   │   ├── IController          # 窗体控制器
│   │   ├── PetLoader            # 宠物图形加载器
│   │   └── GameSave             # 游戏存档
│   ├── Graph/                    # 图形渲染
│   │   ├── IGraph               # 动画接口
│   │   └── GraphCore            # 动画核心
│   └── ...
│
├── VPet-Simulator.Windows/       # 桌面端应用
│   ├── MainWindows/              # 主窗口
│   ├── Function/                 # 功能代码
│   └── WinDesign/               # UI设计
│
└── VPet-Simulator.Tool/          # MOD制作工具
```

## 三、角色适配步骤

### 3.1 准备角色资源

```
泽小安角色包/
├──泽小安/
│   ├── icon.png                  # 512x512 角色图标
│   ├── vpet/                     # 动画资源文件夹
│   │   ├── Happy/               # 开心状态动画
│   │   │   ├── nomal/           # 普通动画
│   │   │   ├── touchhead/       # 摸头
│   │   │   ├── lift/            # 提起
│   │   │   └── ...
│   │   ├── Normal/              # 普通状态动画
│   │   ├── Weak/                # 虚弱状态动画
│   │   └── Ill/                 # 生病状态动画
│   └── vpetinfo                 # 角色配置文件
```

### 3.2 动画资源规格

| 状态 | 动画类型 | 帧数建议 | 说明 |
|------|---------|---------|------|
| Happy | nomal | 15-30帧 | 开心待机 |
| Happy | touchhead | 10-20帧 | 被摸头 |
| Happy | lift | 10-15帧 | 被提起 |
| Normal | nomal | 15-30帧 | 普通待机 |
| Weak | nomal | 15-30帧 | 虚弱待机 |
| Ill | nomal | 15-30帧 | 生病待机 |

**图片规格**：
- 尺寸：根据角色设计（建议 512x512 以内）
- 格式：PNG（透明背景）
- 帧率：12-15 fps

### 3.3 配置文件 (vpetinfo)

```ini
[info]
name = 泽小安
author = 你的名字
desc = 一个戴耳机的可爱Q版角色
version = 1.0

[graph]
# 动画路径配置
happy = vpet/Happy
normal = vpet/Normal
weak = vpet/Weak
ill = vpet/Ill

[level]
# 初始属性值
strength = 100
food = 100
happy = 100
smile = 0
talk = 0
```

## 四、适配方案

### 方案 A：MOD 方式（推荐）

**优点**：不修改 VPet 源码，易于更新

**步骤**：
1. 使用 VPet.ModMaker 制作角色 MOD
2. 打包泽小安角色资源
3. 用户安装 MOD 即可使用

**工具**：
- VPet.ModMaker: https://github.com/LorisYounger/VPet.ModMaker

### 方案 B：Fork 定制

**优点**：深度定制，可修改核心功能

**步骤**：
1. Fork VPet-Simulator.Windows
2. 修改默认角色配置
3. 定制 UI 主题
4. 重新打包发布

### 方案 C：Core 库集成

**优点**：嵌入到自己的 WPF 应用

**步骤**：
1. 引用 VPet-Simulator.Core NuGet 包
2. 创建自定义窗口
3. 加载泽小安角色
4. 实现自定义交互

## 五、推荐方案

**采用方案 A（MOD方式）+ 方案 C（Core集成）**

1. **短期**：制作泽小安角色 MOD，用户直接安装使用
2. **长期**：基于 Core 库开发独立的泽小安桌宠应用
